##  Objetivo del Proyecto
Desarrollar una aplicación web en Python utilizando Flask, estructurada de forma modular, que permita el preprocesamiento de un dataset, la implementación de un modelo sencillo de Machine Learning, el manejo adecuado de excepciones y la visualización interactiva de los resultados utilizando Plotly.

---

##  Características Principales

###  **Funcionalidades Implementadas**
1. **Carga de Datos CSV** mediante interfaz web
2. **Preprocesamiento Automático**:
   - Manejo de valores nulos
   - Normalización min-max
   - Separación de características y target
3. **Modelo de Machine Learning**:
   - Regresión Lineal con scikit-learn
   - División train/test (80/20)
   - Métricas: MAE y R²
4. **Visualización Interactiva**:
   - Gráficos con Plotly
   - Comparación real vs predicción
   - Descarga en formato PNG
5. **Manejo de Excepciones**:
   - Excepciones personalizadas
   - Validación de archivos
   - Mensajes de error amigables

###  **Aspectos Técnicos**
- **Arquitectura Modular**: 4 módulos independientes
- **Interfaz Web Responsive**: HTML5 + CSS3 + JavaScript
- **Framework Web**: Flask
- **ML Framework**: scikit-learn
- **Visualización**: Plotly
- **Despliegue**: Google Colab + ngrok
